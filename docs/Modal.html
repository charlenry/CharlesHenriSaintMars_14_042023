Class: Modal

Modal(displayState, handleClose, message, bgColor, textColor, btnColor) → {JSX.Element}

new Modal(displayState, handleClose, message, bgColor, textColor, btnColor) → {JSX.Element}

This is defining a functional component called Modal that takes in six props,displayState, handleClose, message, bgColor, textColor and btnColor using destructuring syntax. The component returns a JSX element that renders a modal with a message and a close button.

This modal component can be style in the Modal.scss file.

Usage : <Modal displayState={modalDisplayState} handleClose={handleCloseModal} message="Employee Created!" bgColor="" textColor="" btnColor="" />

Component's Hierarchy: CreateEmployee > Modal

Parameters:
Name Type Description
displayState string

The open or closed state of the modal. It can be manage with a local state like in the example above. The two values are block and none. The modal is Hidden by default.
handleClose callback

The callback that handles the close button of the modal
message string

The message to display in the modal
bgColor string

The background color of the modal (optional)
textColor string

The text color of the message (optional)
btnColor string

The basic color of the close button (optional)
